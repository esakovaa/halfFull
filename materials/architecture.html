Safety
🛡️
Safety Layer
Every LLM output is validated and rewritten before reaching the user. Three independent checks run in sequence.
Hard rules (code)
No diagnosis statements · "May suggest / could indicate" framing enforced · Immutable fields never overwritten · Disclaimer on every report
Schema validation
JSON schema check on all LLM output · Auto-repair for truncated responses · Falls back to mock/offline on failure
Tone rewrite
Secondary Groq pass (Llama 3.1-8B) rewrites alarming phrasing · Sensitive keys never logged · Audit trail in Supabase
